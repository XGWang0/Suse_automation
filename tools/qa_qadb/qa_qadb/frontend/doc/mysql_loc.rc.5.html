EXAMPLE

              sql_host=qadb.suse.de
              sql_db=qadb
              sql_user=hamsta-default
              sql_pwd=

DESCRIPTION

       sql_host
              hostname of the computer with MySQL server


       sql_db MySQL database name


       sql_user
              MySQL login


       sql_pwd
              MySQL password
