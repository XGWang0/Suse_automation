SYNOPSIS

       location_detect_impl.pl

DESCRIPTION

       location_detect_impl.pl  guesses  machine's  location.  The location is
       based on the output of ifconfig

       This code is for internal use only, in general, you  should  use  loca-
       tion.pl instead.
