SYNOPSIS

       modify_xml.pl infile -m modfile outfile
       modify_xml.pl -c -m modfile outfile

DESCRIPTION

       This script merges AutoYaST template with a modfile.  XML elements from
       modfile that do not exist in infile are added, the  existing  ones  are
       merged.

       Result  is  then  mostly  used as an XML for setupgrubforinstall(8) and
       AutoYaST.  But theoretically it could have other applications as well.
