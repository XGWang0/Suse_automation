SYNOPSIS

       location.pl

DESCRIPTION

       location.pl  guesses  machine's location.  The location returned is one
       of following: cz , de , cn , us , or (unknown) if the detection fails.

       The location is guessed by reading IP address of local  network  inter-
       faces.   If  the machine has more than one network interfaces, the last
       matching one is used.  See the Perl module detect.pm for more  details.
       Update the heuristics at the same place.

BUGS

       It is just a simple heuristics.
