SYNOPSIS

       product.pl [-v] [-d]

DESCRIPTION

       product.pl  guesses  current machine's product and release, and outputs
       it in the format <product>-<release> , suitable for  qa_db_report.pl  .
       The  script  was  written to allow automatical QADB submissions without
       any user interaction.

       The product is guessed by reading the  output  of  /etc/suse/release  ,
       zypper  -sl  ,  and  /var/lib/YaST2/install.inf .  Candidate products /
       releases are checked against QADB via HTTP.  Best  matching  product  /
       release is printed to STDOUT.

OPTIONS

       -v     Verbose  mode.   Prints  a short summary of the detected product
              and release, and sends it to STDERR.

       -d     Disables QADB use.  By default, the script queries QADB for sup-
              ported products and releases.  When this flag is specified, only
              hardcoded (and possibly outdated) values are used instead.
