Associative array where keys are part IDs und values are another associative array where keys are element names and values are element values.

Example
Example structure: 
$elements = array(
0 => array(
"Description" => "36: None 00.0: 11300 Partition",
"Device File" => "/dev/sda1"
),
1 => array(
"Description" => "37: None 00.0: 11300 Partition",
"Device File" => "/dev/sda2"
)
