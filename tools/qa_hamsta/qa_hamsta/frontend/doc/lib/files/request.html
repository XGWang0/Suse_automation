    This file contains some function to access the parameters passed to the script via the $_REQUEST array.

    These functions should be used in favor of directly accessing the $_REQUEST array or the other superglobals like $_POST and $_GET.

    Their main advantage is that 1) they check if the parameter exists 2) they make sure the parameter is of the expected data type
