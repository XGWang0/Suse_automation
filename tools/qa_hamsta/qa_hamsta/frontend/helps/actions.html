Selected Machine Actions

To send some action to machines, you can either use the "Actions" column in the machines table, or select the machines that you want to work with (using the check-boxes on the left) and then select an action from the dropdown.

The following actions are available for machines that you have selected:

  • Send job - run some tests, or some other pre-defined system tasks
  • Edit/reserve - modify system information like serial console details or who is using the machine and for what purpose
  • Reinstall - perform a fully-automated installation of the system
  • Add to group - group machines together
  • Remove from group - ungroup machines
  • Delete - remove machines from list
  • AutoPXE - install a broken installation, or a raw server
